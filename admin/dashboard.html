Overview

Dashboard Overview

Welcome to your Portfolio Admin Panel. Edit content in any section on the left, then click the 🚀 Publish button at the top-right to push all changes live. Changes are not visible on your portfolio until you Publish.

Quick Actions

How This Works

1
Edit your content
Use the sidebar to navigate to any section — Hero, About, Skills, Experience, etc. Edit and save each one.
2
Upload media if needed
Go to Media Library to upload profile photos, project thumbnails, or gallery images to R2 cloud storage.
3
Click Publish
When you're happy with your changes, click the 🚀 Publish button at the top-right. Your live portfolio will update within seconds.
